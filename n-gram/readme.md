### process data, train and evaluate model
```
usage: ngram.py [-h] --name NAME [--N N] [--batch_size BATCH_SIZE]
                [--epochs EPOCHS]

optional arguments:
  -h, --help            show this help message and exit
  --name NAME           Name of model
  --N N                 Order of n-gram
  --batch_size BATCH_SIZE
                        Number of examples to process in a batch
  --epochs EPOCHS       Number of training epochs

```
